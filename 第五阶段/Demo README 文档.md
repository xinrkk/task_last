## 项目名称

简易计算器

------

## 一、程序简介

这是一个使用 Go 语言编写的控制台计算器程序。
 用户可以在终端中选择加、减、乘、除运算，并输入两个数字进行计算。

------

## 二、功能介绍

程序包含以下功能：

1. 加法运算
2. 减法运算
3. 乘法运算
4. 除法运算（含除 0 判断）
5. 退出程序

------

## 三、运行方法

进入程序目录，执行：

```
go run main.go
```

或：

```
go build main.go
./main
```

Windows 下：

```
main.exe
```

------

## 四、使用方法

操作步骤：

1. 输入功能编号
2. 输入两个数字
3. 查看结果

程序会循环运行，直到选择退出。

------

## 五、使用到的 Go 基础知识点

本程序使用了以下 Go 基础知识：

1. package main 和 main 函数
2. fmt 包（输入输出）
3. 变量声明（var）
4. 基本数据类型（int、float64）
5. for 无限循环
6. if 条件判断
7. switch 语句
8. break、continue
9. 格式化输出（Printf）

10. 简单错误处理（除 0 判断）
